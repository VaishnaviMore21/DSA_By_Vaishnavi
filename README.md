# DSA By Vaishnavi

> A structured Data Structures and Algorithms repository for consistent interview-focused practice.

---

## About This Repository

This repository organizes DSA problems topic-wise and keeps each solution path easy to revise.
Each problem is maintained with multiple approaches so you can compare trade-offs clearly.

### Goals

- Build strong fundamentals through regular practice.
- Progress from simple to optimized solutions.
- Maintain clean, revision-friendly notes and code.

## Problem Format

Most problem folders follow a consistent format:

```text
Problem_Name/
	BruteForce.java
	Better.java
	Optimal.java
	Notes.md
```

This structure makes it easy to compare time and space complexity from naive to optimized approaches.

---

## How To Use This Repo

1. Start with any topic folder.
2. Pick a problem and attempt it first.
3. Compare your logic with BruteForce, Better, and Optimal approaches.
4. Read Notes to revise intuition, edge cases, and complexity.
