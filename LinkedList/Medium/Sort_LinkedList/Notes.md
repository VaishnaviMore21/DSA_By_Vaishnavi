# Sort_LinkedList

## Problem statement
Implement Sort_LinkedList using LinkedList patterns.

## Intuition
Use node pointer operations and traversal carefully to preserve list structure.

## Brute force
Extract values, sort array, rebuild list.

## Better
Insertion sort on linked list.

## Optimal
Merge sort on linked list is optimal.

## Dry run
Take sample list and trace pointer updates step by step at each node change.

## Edge cases
Empty list, single node list, and minimal valid length inputs.

## Time complexity
O(n log n)

## Space complexity
O(1) extra for iterative pointer-based methods unless hashing/recursion used.

## Interview notes
Split using slow/fast pointer.
