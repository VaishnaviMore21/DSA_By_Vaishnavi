# Search_In_LinkedList

## Problem statement
Implement Search_In_LinkedList using LinkedList patterns.

## Intuition
Use node pointer operations and traversal carefully to preserve list structure.

## Brute force
Linear search from head.

## Better
No better asymptotic approach in linked list.

## Optimal
Linear scan is optimal in unsorted list.

## Dry run
Take sample list and trace pointer updates step by step at each node change.

## Edge cases
Empty list, single node list, and minimal valid length inputs.

## Time complexity
O(n)

## Space complexity
O(1) extra for iterative pointer-based methods unless hashing/recursion used.

## Interview notes
Early exit on first match improves practical runtime.
